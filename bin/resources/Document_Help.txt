This program is a unit of measurement converter.

The menu options are very straight: 

* From the "File" menu, one can choose among some available actions that draw shapes in the background, and finish the program;
* From the "Help" menu, one can choose see standard texts: "Help", "Disclaimer", and "About".


Technical stuff:

* The program was written in Java 1.8 and runs on most of desktop and notebook operating systems.
